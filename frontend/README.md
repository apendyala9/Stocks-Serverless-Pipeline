# Frontend Package

This package contains the React SPA for the Stocks Serverless Pipeline dashboard.

Canonical project documentation lives in the root README:

- [`../README.md`](../README.md)

## Common Commands

- `npm run dev` - run Vite dev server
- `npm run build` - build production bundle to `dist/`
- `npm run preview` - preview production build locally
- `npm run test` - run frontend tests
- `npm run lint` - run ESLint

## Environment

Use [`./.env.example`](./.env.example) as the starting point for local configuration.
